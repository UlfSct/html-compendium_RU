• <audio>
Отвечает за работу с аудиозаписями, брать только .wav формат. 

autoplay - звук начинает играть сразу после загрузки страницы.
<audio autoplay></audio>

controls - добаляет панель управления к аудиофайлу.
<audio controls></audio>	

loop - повторяет воспроизведение звука после завершения аудио.
<audio loop></audio>

preload - используется для загрузки аудио при загрузке веб-страницы.
<audio preload="none | metadata | auto"></audio>
metadata - только служебная информация (длительность и др.)
auto - всё сразу

src - путь к воспроизводимому файлу, для этой же цели используется <source>.
<audio src="music.wav"></audio>


• <video>
Добавляет и воспроизводит видео.

autoplay - видео начинает воспроизводиться автоматически.

controls - добавляет панель управления к видеоролику.

height - задаёт высоту видео.

loop - видео автоматически зацикливается.

poster - указывает адрес картинки, которая будет показываться, пока видео не доступно или не воспроизводится.

preload - используется для загрузки видео вместе с веб-страницей. Игнорируется при autoplay.
<video preload="none | metadata | auto"></video>
none - не загружать видео.
metadata - загрузить только служебную информацию (размер, первый кадр, продолжительность и др.).
auto - загрузить видео целиком при загрузке страницы.
По умолчанию - none.

src - указывает на путь к воспроизводимому видео.

width - задаёт ширину тега видео.


• <source> - вставляет аудио или видео внутри соответствующих тегов.
<audio>
   <source src="">
</audio>

media - определяет устройство, для которого будет воспроизводиться аудио/видео.
<source media="all | braille | handheld | print | screen | speech | projection | tty | tv">
all - все устройства.
braille - устройства для слепых людей.
handheld - устройства с малой шириной экрана.
print - печатающее устройство вроде принтера.
screen - экран монитора.
speech - речевые синтезаторы.
projection - проектор.
tty - телетайпы, терминалы, портативные устройства с ограниченными возможностями экрана.
Для них не должны использоваться пиксели в качестве единиц измерения.
tv - телевизоры. 

src - определяет ссылку на файл.

type - указывает MIME-тип источника, а также аудио и видеокодек.
