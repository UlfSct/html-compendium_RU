• <embed>
Предназначен для загрузки файлов, которые бразуер исходно не понимает.
(странные форматы аудио и видео).

height - устанавливает высоту объекта

width - устанавливает ширину объекта

src - указывает на путь к файлу

type - указывает на MIME-тип файла


• <progress>
Используется для отображения прогресса завершённой задачи. 
Значение изменяется через JavaScript.

value - текущее значение прогресса.

max - максимальное значение прогресса.


• <meter>
Используется для графического отображения значения в некотором диапазоне.

value - устанавливает значение

min - задаёт минимальное значение

max - задаёт максимальное значение

low - определяет предел низкого значения
 
high - определяет предел высокого значения

optimum - определяет оптимальное значение


• <canvas>
Создаёт обалсть, в которой можно рисовать при помощи js.

height - задаёт высоту холста (150 по умолчанию).

width - задаёт ширину холста (300 по умолчанию).

• <iframe>
Создаёт плавающий фрейм, который находится внутри обычного документа, позволяет загружать в 
область заданных размеров любые документы.

height - задаёт высоту фрейма

width - задаёт ширину фрейма

name - задаёт имя фрейма, необходимое для отправки в него содержимого

sandbox - позволяет установить ряд ограничений на контент, загружаемыый во фрейме.
<iframe ... sandbox="allow-some-origin | allow-top-navigation |allow-forms | allow scripts></iframe>
allow-some-origin - разрешает загружать содержимое фрейма. Используется для безопасного открытия контента.
allow-top-navigation - позволяет открывать ссылки фрейма в родительском документе.
allow-forms - позволяет содержимому фрейма отправлять формы.
allow-scripts - разрешает выполнение скриптов, вспывающие окна при этом запрещены.
Допустимо писать несколько значений через пробел, отсутствие занчения - запрет действия.
При использовании allow-scripts и allow-some-origin, когда исходный и загружаемый документ одного
происхождения, атрибут sandbox игнорируется.

src - указывает адрес файла, который будет загружаться в фрейм.
