• <p>
Определяет текстовый абзац.

Только универсальные атрибуты и события.


• <h1>, <h2>, <h3>, <h4>, <h5>, <h6>
Отвечают за иерархию разделов сайта (означают важность заголовка от главного (1) к наследующим (6)).

Только универсальные атрибуты и события.


• <wbr>
Указывает место в тексте, где допустимо сделать перенос строки.


• <br>
Устанавливает перевод строки в том месте, где находится тег.


• <pre>
Определяет блок предварительно форматированного текста. Сохраняет все пробелы между словами.
Внутри допустимо применять любые теги кроме следующих: <img>, <small>, <sub>, <sup>.

Только универсальные атрибуты и события.


• <sub>
Отображает текст в нижнем индексе. Есть также vertical-align: sub.

Только универсальные атрибуты и события.


• <sup>
Отображает текст в верхнем индексе. Есть также vertical-align: sup.

Только универсальные атрибуты и события.


• <em>
Делает текст курсивом.
<em>Хуй</em>

только универсальные атрибуты и события


• <strong>
Выделяет шрифт жирным начертанием.

Только универсальные атрибуты и события.


• <s> 
Отображает текст перечёркнутым.


• <tt>
Отображает текст моноширинным текстом.


• <bdi>
Позволяет фиксировать направление текста на странице.
dir - выбирает направление фиксации тега <p>.
<p dir="rtl"><bdi>שבת</bdi> - суббота</p>

только универсальные атрибуты и события.


• <bdo>
Устанавливает направление вывода текста (преимущественно гле чтение справа налево).

dir - задаёт направление фиксации (rtl | ltr).
<p><bdo dir="rtl">А роза упала на лапу Азора</bdo></p>


• <abbr> 
Указывает, что последовательность символов является аббревиатурой.

title - добавляет всплывающую расшифровку аббревиатуры.
<abbr title="Реальный Хуй Тупо Утопия">РХТУ</abbr>

+ универсальные атрибуты и события.


• <blockquote>
Предназначен для выделения очень длинных цитат.
По дефолту отображается как выровненный блок с отступами слева и справа, а так же отбивкой сверху и снизу.

cite - указывает на источник цитаты, больше необходим для алгоритмов поиска.
<blockquote cite="vk.com">Телеграм - сила</blockquote>


• <cite>
Помечает текст как цитату или сноску на другой материал. Является ещё одной семантической приблудой.
<p><q>...</q></p>
<p><cite>ccылка</cite></p>

Только универсальные атрибуты и события.


• <code>
Предназначен для отображения кода на html-странице. Семантический, мелкий курсив.
Не учитывает пробелы и переносы в отличие от <pre>.

Только универсальные атрибуты и события.


• <del>
Используется для выделения текста, который был удалён в новой версии документа.
Семантическая приблуда, текст обычно перечёркнутый.

cite - указывает ссылку на документ с причиной редактирования.

datetime - устанавливает дату и время редактирования.
<del datetime="2011-11-05T23:31:05+07:00">Текст</del>


• <dfn>
Семантика текста для новых определений (курсив).

Только универсальные атрибуты и события.


• <ins>
Семантика текста, добавленного в новую версию документа. Обычно подчёркнутый.

Только универсальные атрибуты и события.


• <kbd>
Семантика текста для обозначения клавиш, команд или тегов. Обычно моноширинным текстом.

Только универсальные атрибуты и события.


• <mark>
Семантика текста для выделения важных частей. По умолчанию жёлтый фон


• <output>
Семантика текста, отвечающая за место вывода информации. Обычно находится внутри form.


• <q>
Семантика текста. Используется для выделения цитат, по-умолчанию в кавычках.

cite - ссылка на источник цитаты.

+ универсальные атрибуты и события.


• <samp>
Используется для отображения текста, который является результатом выводв программы или скрипта.

Только универсальные атрибуты и события.


• <time>
Семантика текста. Указывает внутри значение даты и|или времени (также можно атрибутом datetime).

datetime - устанавливает дату, время или оба значения для текста в формате ISO 8601.
https://ru.wikipedia.org/wiki/ISO_8601


• <var>
Семантика текста для выделения переменных программ. По умолчанию - курсив.

Только универсальные атрибуты и события.


• <a>
Предназначен для создания ссылок.

accesskey - активация ссылки с помощью комбинации клавиш.
<a accesskey="c">...</a>
В разных браузерах работает по-разному:
Chrome: Alt + c
Opera: Shift + Esc + S
и.т.д.

download - предлагает скачать указанный по ссылке файл.
<a download>Ссылка</a>
<a href="Ссылка" download>...</a>

href - задаёт адрес документа, на который необходимо перейти.
<a href="URL">...</a>

hreflang - идентефицирует язык текста по ссылке.
<a href="URL" hreflang="ru">..</a>
Необходим для перемещения на страницу с конкретным языком, когда сайт международный

rel - отношения между ссылаемым и текущим документами.
<a rel="строка">...</a>
answer - ответ на вопрос
chapter - раздел или глава текущего докуменат
и.т.д. http://wiki.whatwg.org/wiki/RelExtensions

tabindex - определяет последовательность перехода между ссылками при нажатии tab.
<a tabindex="число">...</a>

target - имя окна или фрейма, куда браузер будет загружать документ.
<a target="имя окна">...</a>
_blank - в новом окне
_self - в текущем окне
_parent - в фрейм-родитель, если такого нет, то _self
_top - отменяет все фреймы и загружает в полном окне браузера, если фреймов нет - _self

title - добавляет всплывающую подсказку к тексту ссылки.
<a title="текст">...</a>

type - указывает MIME-тип документа, на который ведёт ссылка.
<a href="URL" type="MIME-тип">...</a>
http://htmlbook.ru/html/value/mime

+ универсальные атрибуты и события
