• <!DOCTYPE html>
Предназначен для указания типа текущего документа, для меня всегда html.


• <html>
Является контейнером, заключающим в себе всё содержимое страницы.

manifest - реализует механизм кэширование, то есть позволяет создавать оффлайн-приложения.
При первой загрузке страницы просит сохранить данные, а заетм обращаться к ним при необходимости.
Файл кэша должен отдаваться с заголовком text/cache-manifest.
<html manifest="123456.cache>
   ...
</html>


• <head>
Предназначен для хранения элементов, цель которых - помочь браузеру в работе с данными.
Из всего содержимого отображается только <title>

Только универсальные атрибуты.


• <title>
Задаёт заголовк страницы. Используется один раз внутри <head>.


• <meta>
Определяет метатеги, которые нужны для хранения информации, предназначенной браузеру.
Всегда расположен в <head>.

charset - устанавливает кодировку текста

name - устанавливает идентификатор метатега для пары имя=значение.
Не допускается одновременно с http-equiv.

http-equiv - аналогичен name, но браузеры преобразовывают его значение в content в формат заголовка http.
Не допускается одновременно с name.

content - устанавливает значение атрибута, заданного с помощью name или http-equiv.


• <base>
Инструктирует браузер о базовом адресе текущего документа для всех относительных адресов.

href - базовый адрес полного пути.
<base href="http://www.hui.ru/page.html">

target - задаёт имя контекста, которое применяется для всех <a> и <form>.
<base target="имя окна">
_blank - в новом окне
_self - в текущем окне
_parent - в фрейм-родитель, если такого нет, то _self
_top - отменяет все фреймы и загружает в полном окне браузера, если фреймов нет - _self


• <link>
Устанавливает связь с внешним документом типа стилей или шрифтов.
Всегда размещается в тэге <head>.

href - указывает путь к необходимому файлу.

media - определяет устройство, для которого применять данные стили.
<link media="all | braille | handheld | print | screen | speech | projection | tty | tv">
all - все устройства.
braille - устройства для слепых людей.
handheld - устройства с малой шириной экрана.
print - печатающее устройство вроде принтера.
screen - экран монитора.
speech - речевые синтезаторы.
projection - проектор.
tty - телетайпы, терминалы, портативные устройства с ограниченными возможностями экрана.
Для них не должны использоваться пиксели в качестве единиц измерения.
tv - телевизоры. 

rel - определяет тип подключаемого документа.

type - сообщает браузеру MIME-тип документа.

+ универсальные атрибуты.


• <style>
Применяется для определения стилей веб-страницы. Обязательно внутри <head>.

media - определяет устройство, для которого применять данные стили.
<source media="all | braille | handheld | print | screen | speech | projection | tty | tv">
all - все устройства.
braille - устройства для слепых людей.
handheld - устройства с малой шириной экрана.
print - печатающее устройство вроде принтера.
screen - экран монитора.
speech - речевые синтезаторы.
projection - проектор.
tty - телетайпы, терминалы, портативные устройства с ограниченными возможностями экрана.
Для них не должны использоваться пиксели в качестве единиц измерения.
tv - телевизоры. 

type - сообщает бразуеру, какой синтаксис использовать для определения стилей.


• <script>
Предназначен для описания скриптов. Может содержать ссылку на программу или её текст.

async - асинхронность выполнения скрипта.
<script async></async>

defer - откладывает выполнение скрипта, пока вся страница не будет загружена.
<script defer></script>

src - указывает путь на .js файл

type - указывает MIME-тип для определённого языка


• <noscript>
Показывает своё содержимое, если браузер не поддерживает работу со скриптами.


• <body>
Предназначен для хранения содержимого веб-страницы.

только универсальные атрибуты и события.


• <hr>
Рисует горизонтальную линию.

Только универсальные атрибуты и события.


• <!-- -->
Предназначен для добавления комментариев внутри html-разметки.
