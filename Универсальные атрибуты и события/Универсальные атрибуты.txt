class - задаёт класс стилей, который позволяет связать элемент страницы с его оформлением.

id - задаёт идентификатор, который позволяет связать элемент страницы с его оформлением / скриптами.

lang - указывает язык внутри элемента страницы.

dir="rtl | ltr" - задаёт направление отображения текста.

contenteditable="true | false" - сообщает, что элемент доступен для редактирования.

title - описывает содержимое элемента в виде всплывающей подсказки.

style - описывает стили для определённого блока.

hidden - скрывает содержимое элемента со страницы, при этом элемент доступен через скрипты.
