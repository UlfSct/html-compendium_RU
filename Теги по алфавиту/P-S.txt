• <p>
Определяет текстовый абзац.

Только универсальные атрибуты и события.


• <pre>
Определяет блок предварительно форматированного текста. Сохраняет все пробелы между словами.
Внутри допустимо применять любые теги кроме следующих: <img>, <small>, <sub>, <sup>.

Только универсальные атрибуты и события.


• <progress>
Используется для отображения прогресса завершённой задачи. 
Значение изменяется через JavaScript.

value - текущее значение прогресса.

max - максимальное значение прогресса.


• <q>
Семантика текста. Используется для выделения цитат, по-умолчанию в кавычках.

cite - ссылка на источник цитаты.

+ универсальные атрибуты и события.


• <s> 
Отображает текст перечёркнутым.


• <samp>
Используется для отображения текста, который является результатом выводв программы или скрипта.

Только универсальные атрибуты и события.


• <script>
Предназначен для описания скриптов. Может содержать ссылку на программу или её текст.

async - асинхронность выполнения скрипта.
<script async></async>

defer - откладывает выполнение скрипта, пока вся страница не будет загружена.
<script defer></script>

src - указывает путь на .js файл

type - указывает MIME-тип для определённого языка


• <section>
Семантика div. Отвечает на разделение сайта на секции, содержит заголовок.

Только универсальные атрибуты и события.


• <select>
Позволяет создать выпадающий список. Принимает в себя теги <option>.

accesskey - позволяет перейти к списку по горячим клавишам.

autofocus - список получает фокус сразу после загрузки страницы.
<select autofocus>...</select>

disabled - запрещает взаимодействие со списком.

form - связывает список с формой по её идентефикатору.

multiple - позволяет выбирать несколько значений с помощью Shift и Ctrl.

name - создаёт уникальное имя.

required - задаёт обязательность выбора одной из опций.

size - устанавливает высоту выпадающего списка.

tabindex - определяет последовательность перехода к элементу по Tab.

+ универсальные атрибуты и события.


• <span>
В отличие от div, p, table явялется строчным типом, то есть можно задать другой стиль 
для чего-либо, при этом не потеряв уже настроенное расположение элементов.

Только универсальные атрибуты и события.


• <source> - вставляет аудио или видео внутри соответствующих тегов.
<audio>
   <source src="">
</audio>

media - определяет устройство, для которого будет воспроизводиться аудио/видео.
<source media="all | braille | handheld | print | screen | speech | projection | tty | tv">
all - все устройства.
braille - устройства для слепых людей.
handheld - устройства с малой шириной экрана.
print - печатающее устройство вроде принтера.
screen - экран монитора.
speech - речевые синтезаторы.
projection - проектор.
tty - телетайпы, терминалы, портативные устройства с ограниченными возможностями экрана.
Для них не должны использоваться пиксели в качестве единиц измерения.
tv - телевизоры. 

src - определяет ссылку на файл.

type - указывает MIME-тип источника, а также аудио и видеокодек.


• <strong>
Выделяет шрифт жирным начертанием.

Только универсальные атрибуты и события.


• <style>
Применяется для определения стилей веб-страницы. Обязательно внутри <head>.

media - определяет устройство, для которого применять данные стили.
<source media="all | braille | handheld | print | screen | speech | projection | tty | tv">
all - все устройства.
braille - устройства для слепых людей.
handheld - устройства с малой шириной экрана.
print - печатающее устройство вроде принтера.
screen - экран монитора.
speech - речевые синтезаторы.
projection - проектор.
tty - телетайпы, терминалы, портативные устройства с ограниченными возможностями экрана.
Для них не должны использоваться пиксели в качестве единиц измерения.
tv - телевизоры. 

type - сообщает бразуеру, какой синтаксис использовать для определения стилей.


• <sub>
Отображает текст в нижнем индексе. Есть также vertical-align: sub.

Только универсальные атрибуты и события.


• <summary>
Устанавливает заголовок для тега <details>.


• <sup>
Отображает текст в верхнем индексе. Есть также vertical-align: sup.

Только универсальные атрибуты и события.
