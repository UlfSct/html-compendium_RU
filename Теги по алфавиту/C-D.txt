height - задаёт высоту холста (150 по умолчанию).

width - задаёт ширину холста (300 по умолчанию).

• <caption>
Предназначен для создания заголовка в таблице. Пишется сразу после открывающего <table> 

Только универсальные атрибуты и события


• <cite>
Помечает текст как цитату или сноску на другой материал. Является ещё одной семантической приблудой.
<p><q>...</q></p>
<p><cite>ccылка</cite></p>

Только универсальные атрибуты и события


• <code>
Предназначен для отображения кода на html-странице. Семантический, мелкий курсив.
Не учитывает пробелы и переносы в отличие от <pre>.

Только универсальные атрибуты и события.


• <col> 
Задаёт ширину и другие характеристики одной или нескольких колонок таблицы.
При наличии этого тега, браузер показывает содержимое таблицы не дожидаясь загрузки страницы.

span - количество колонок, к которым применятся данные характеристики.
<table><col span="5"><tr>...


• <colgroup>
Предназначен для задания ширины и стиля одной или нескольких колонок таблицы.
<colgroup>
	<col ...>
	<col ...>
	<col ...>
</colgroup>

span - количество колонок, к которым применятся данные характеристики.

width - задаёт ширину колонки.
<colgroup width="число">


• <datalist>
Создаёт список вариантов, которые можно выбирать при наборе в текстовом поле.
Изначально скрыт, виден только при получении полем фокуса или наборе текста.
Список <datalist> связывается через id с помощью атрибута list у <input>
<input list="hui">
<datalist id="hui">
 <option value="hui1">
 <option value="hui2">
</datallist>


• <dl>, <dt>, <dd>
Предназначены для создания списка определений.
<dl> - контейнер, аналогичен <li>.
<dt> - тег, создающий термин.
<dd> - тег, задающий определение термина.
<dl>
  <dt>Термин 1</dt>
  <dd>гавно, залупа или просто пенис</dd>
  ...
</dl>

Только универсальные атрибуты и события


• <del>
Используется для выделения текста, который был удалён в новой версии документа.
Семантическая приблуда, текст обычно перечёркнутый.

cite - указывает ссылку на документ с причиной редактирования.

datetime - устанавливает дату и время редактирования.
<del datetime="2011-11-05T23:31:05+07:00">Текст</del>


• <details>.
Используется для хранения информации типа аккордеона.
С помощью тега <summmary> можно задать шапку аккордеона.
<details>
<summary>окси</summary>
гавно, залупа...
</details>

open - по дефолту показывает содержимое блока.
<details open> </details>


• <dfn>
Семантика текста для новых определений (курсив).

Только универсальные атрибуты и события


• <div>
Базовый блочный элемент для выделения объекта в класс или id.
