• <label>
Устанавливает связь между текстом и элементом формы.
Такая связь необходима для перемещения к полю при нажатии на текст.

accesskey - доступ к элементам формы по горячим клавишам.

for - позволяет ссылаться на id определённого поля ввода.
<input id="penis><label for="penis">хуй</label>


• <legend>
Отвечает за создание заголовка группы элементов формы <fieldset>
<fieldset>
   <legend>типа группа элементов</legend>
      <input>
      ...
</fieldset>

accesskey - перейти к группе элементов формы по горячим клавишам.

title - добавляет поясняющий всплывающий текст.

+ универсальные атрибуты и события.


• <li> Определяет отдельный элемент списка

value - устанавливает число, с котором будет начинаться список (только для нумерованных <ol>).

+ универсальные атрибуты и события.


• <link>
Устанавливает связь с внешним документом типа стилей или шрифтов.
Всегда размещается в тэге <head>.

href - указывает путь к необходимому файлу.

media - определяет устройство, для которого применять данные стили.
<link media="all | braille | handheld | print | screen | speech | projection | tty | tv">
all - все устройства.
braille - устройства для слепых людей.
handheld - устройства с малой шириной экрана.
print - печатающее устройство вроде принтера.
screen - экран монитора.
speech - речевые синтезаторы.
projection - проектор.
tty - телетайпы, терминалы, портативные устройства с ограниченными возможностями экрана.
Для них не должны использоваться пиксели в качестве единиц измерения.
tv - телевизоры. 

rel - определяет тип подключаемого документа.

type - сообщает браузеру MIME-тип документа.

+ универсальные атрибуты.


• <main>
Семантика div для главного контента. Не должен включать прочие блоки типа шапки, подвала, боковой панели и.т.п.


• <map>
Служит контейнером для элементов <area>. Его цель - связывание карты с изображением <img>.

name - имя карты изображения.


• <mark>
Семантика текста для выделения важных частей. По умолчанию жёлтый фон


• <meta>
Определяет метатеги, которые нужны для хранения информации, предназначенной браузеру.
Всегда расположен в <head>.

charset - устанавливает кодировку текста

name - устанавливает идентификатор метатега для пары имя=значение.
Не допускается одновременно с http-equiv.

http-equiv - аналогичен name, но браузеры преобразовывают его значение в content в формат заголовка http.
Не допускается одновременно с name.

content - устанавливает значение атрибута, заданного с помощью name или http-equiv.


• <meter>
Используется для графического отображения значения в некотором диапазоне.

value - устанавливает значение

min - задаёт минимальное значение

max - задаёт максимальное значение

low - определяет предел низкого значения
 
high - определяет предел высокого значения

optimum - определяет оптимальное значение


• <nav>
Семантика div для навигации. Запрещается вкладывать в address.


• <noscript>
Показывает своё содержимое, если браузер не поддерживает работу со скриптами.


• <ol>
Устанавливает нумерованный список.

type - устанавливает тип маркеров.
<ol type="A | a | I | i | 1">...</ol>
A - заглавные латинские буквы.
а - строчный латинские буквы.
I - заглавные римские цифры.
i - строчные римские цифры.
1 - арабские цифры.

reversed - задаёт обратный порядок списка
<ol reversed>...</ol> - элементы <li> выведутся в обратном порядке.

start - номер, с которого будет начинаться список
<ol start="3">...</l> 


• <optgroup>
Контейнер для группировки элементов <option>. Не выделяется, но все наследующие сдвигаются вправо.
<select>
   <optgroup>
      <option>...</option>
      ...
   </optgroup>
   <optgroup>
   ...
</select>

disabled - блокирует доступ к группе, обозначая их другим цветом.
<optgroup disabled>...</optgroup>

label - текст, который будет входить в список в виде заголовка группы.

+ универсальные атрибуты и события.


• <option>
Определяет отдельные пункты списка из контейнера <select>.

disabled - блокирует элемент списка для его выбора.

label - предназначен для сокращения длины текста внутри option. 
Пользователю будет показоно содержимое label.

selected - делает текущий элемент списка выделенным.

value - определяет значение для отправки на сервер.

+ универсальные атрибуты и события


• <output>
Семантика текста, отвечающая за место вывода информации.
