• <table>
Контейнер для элементов, определяющих содержимое таблицы.

Только универсальные атрибуты и события.


• <tbody>
Предназначен для хранения одной или нескольких строк таблицы.
В одной таблице может быть множество тегов tbody.

Только универсальные атрибуты и события.


• <td>
Контейнер для содержимого одной ячейки определённой строчки tr.

colspan - устанавливает количество ячеек, которые необходимо объединить по горизонтали.
Если значение 0, то ячейки объединятся до конца таблицы.
Значения выше 1000 считаются ошибочными и превращаются в 0.

rowspan - устанавливает количество ячеек, которые необходимо объединить по вертикали.
Если значение 0, то ячейки объединятся до конца таблицы.
Значения выше 1000 считаются ошибочными и превращаются в 0.

headers - связывает ячейки таблицы с заголовками (для речевых браузеров).
<table>
   <thead>
      <tr><th id="name">Имя</th><th id="surname>Фамилия</th></tr>
   </thead>
   <tbody>
      <tr><td headers="name">Ваня</td><td headers="surname">Иванов</td></tr>
      ...
   </tbody>
</table>

+ универсальные атрибуты и события.


• <textarea>
Элемент формы для написания нескольких строк текста.
В отличие от input, допустимы переносы строк,они сохраняться при передаче данных на отработку.

accesskey - доступ к элементу по горячим клавишам.

autofocus - автоматически устанавливает фокус в текстовое поле.

cols - задаёт ширину поля, значение - количество умещающихся символов в строке.

disabled - отключает текстовое поле.

form - связывает поле с формой по её id.

maxlength - максимальное число символов поля.

name - уникальное имя элемента формы.

placeholder - подсказка внутри поля, которая исчезает при фокусе.

readonly - работает как disabled, но стили при этом сохраняются.

required - обязательность заполнения данного поля формы.

rows - задаёт высоту поля, значение - количество умещающихся символов в столбце.

tabindex - последовательность перехода между элементами формы по Tab.

wrap - работа с переносом текста.
<textarea wrap="soft | hard"></textarea>
soft - текст будет пернесён, но на сервер отправятся только переносы от пользователя.
hard - текст будет перенесён, на сервер отправится со всеми переносами.
По умолчанию - soft.

+ универсальные атрибуты и события.


• <tfoot>
Предназначен для хранения одной или нескольких строк таблицы.
Объявляется до <tbody>, но отображается в конце таблицы.
<table>
   <thead>
      ...
   </thead>
   <tfoot>
      ...
   </tfoot>
   <tbody>
      ...
   </tbody>
</table>

Только универсальные атрибуты и события.


• <th>
Полностью аналогичен <td>, но для заголовков таблицы.

colspan - устанавливает количество ячеек, которые необходимо объединить по горизонтали.
Если значение 0, то ячейки объединятся до конца таблицы.
Значения выше 1000 считаются ошибочными и превращаются в 0.

rowspan - устанавливает количество ячеек, которые необходимо объединить по вертикали.
Если значение 0, то ячейки объединятся до конца таблицы.
Значения выше 1000 считаются ошибочными и превращаются в 0.

scope - связывает между собой заголовки и столбцы таблицы. Аналог headers, но для простых таблиц.
col - заголовок для одного столбца.
colgroup - заголовок для группы столбцов (только с тегом <colgroup>).
<table>
   <thead>
      <tr><th scope="col">Имя</th><th scope="col">Фамилия</th></tr>
   </thead>
   <tbody>
      <tr><td>Ваня</td><td>Иванов</td></tr>
      ...
   </tbody>
</table>

+ универсальные атрибуты и события.

• <video>
Добавляет и воспроизводит видео.

autoplay - видео начинает воспроизводиться автоматически.

controls - добавляет панель управления к видеоролику.

height - задаёт высоту видео.

loop - видео автоматически зацикливается.

poster - указывает адрес картинки, которая будет показываться, пока видео не доступно или не воспроизводится.

preload - используется для загрузки видео вместе с веб-страницей. Игнорируется при autoplay.
<video preload="none | metadata | auto"></video>
none - не загружать видео.
metadata - загрузить только служебную информацию (размер, первый кадр, продолжительность и др.).
auto - загрузить видео целиком при загрузке страницы.
По умолчанию - none.

src - указывает на путь к воспроизводимому видео.

width - задаёт ширину тега видео.


• <wbr>
Указывает место в тексте, где допустимо сделать перенос строки.
